### Registers

| REGISTERS              | DESCRIPTION                      |
| ---------------------- | -------------------------------- |
| `A`, `B`, `C`, `D`     | general-purpose scalar registers |
| `V0`, `V1`, `V2`, `V3` | general-purpose vector registers |
| `X`, `Y`, `Z`, `W`     | scalar elements of `V0`          |
| `IP`                   | instruction pointer              |
| `SP`                   | stack pointer                    |
